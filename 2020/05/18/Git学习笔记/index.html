Git学习笔记

前言

  • 首先推荐一款快速学习Git的工具——Learn Git通过游戏的方式，进行深入学习。
  • 关于Git的详细介绍，参考Git教程

Git常用基础命令

1.Git Init

  git commit命令创建一个空的Git仓库或重新初始化一个现有仓库，即使在现有存储库中运行git init命令也是安全的，它不会覆盖已经存在的东西。

2.Git Add

  git add命令将文件内容添加到索引(将修改添加到暂存区)。也就是将要提交的文件的信息添加到索引库中。该命令可以在提交之前多次执行。它只在运行git add命令时添加指定文件的内容; 如果希望随后的更改包含在下一个提交中，那么必须再次运行git add将新的内容添加到索引。

3.Git Clone

  git clone命令将存储库克隆到新目录中。

4.Git Commit

  git commit命令用于将更改记录(提交)到存储库。将索引的当前内容与描述更改的用户和日志消息一起存储在新的提交中。

5.Git Status

  git status命令用于显示工作目录和暂存区的状态。使用此命令能看到那些修改被暂存到了, 哪些没有, 哪些文件没有被Git tracked到。git status不显示已经commit到项目历史中去的信息。看项目历史的信息要使用git log。

6.Git Reset

  git reset命令用于将当前HEAD复位到指定状态。一般用于撤消之前的一些操作(如：git add,git commit等)

7.Git Branch

  git branch命令用于列出，创建或删除分支。

  • 查看当前有哪些分支
  • 新建一个分支
  • 切换到指定分支
  • 查看本地和远程分支
  • 修改分支名字
  • 删除分支
  • 合并分支
  • 更新远程仓库到本地&&推送分支

8.Git Rm && Git Mv

  git rm命令用于从工作区和索引中删除文件。

  git mv命令用于移动或重命名文件，目录或符号链接。

  目前常用到的命令暂时就这些，如用到其余命令以后再作补充。
